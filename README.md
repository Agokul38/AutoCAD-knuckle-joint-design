# AutoCAD-knuckle-joint-design
3D Knuckle Joint model designed using AutoCAD. Includes detailed mechanical components, assembly, accurate dimensions, and engineering drawings for mechanical design portfolio.
## Overview
This project showcases the design and 3D modeling of a Knuckle Joint using AutoCAD. The model was created following standard mechanical engineering design principles and includes detailed components with accurate dimensions. It serves as a portfolio project demonstrating CAD modeling and engineering drawing skills.

## Features
- 3D Knuckle Joint Assembly
- Individual Part Modeling
- Accurate Dimensions
- Engineering Drawings
- AutoCAD DWG File
- PDF Drawing
